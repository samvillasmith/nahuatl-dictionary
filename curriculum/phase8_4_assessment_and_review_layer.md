# Phase 8.4 — Assessment and Review Layer

## Purpose
Turn the unit corpus into a teachable, testable progression.

## Deliverables
- unit mastery checklist
- end-of-unit quiz
- cumulative review items
- speaking prompts
- writing prompts
- A1 / A2 / B1 / B2 review packs
- answer key scaffold
- scoring/rubric scaffold

## Assessment model
Each unit should include:
- comprehension
- controlled production
- free production
- review and recycling

## Review structure
- immediate unit quiz
- 3-unit cumulative review
- band-end review
- B2 bridge review

## Exit criteria
Every pedagogical unit has:
- quiz items
- review items
- speaking task
- writing task
- mastery checklist
