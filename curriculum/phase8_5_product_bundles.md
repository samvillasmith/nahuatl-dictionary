# Phase 8.5 — Product Bundles

## Purpose
Package the instructional content so it can become a product without re-querying the database.

## Bundle targets
### App bundle
- unit.json
- vocab.json
- constructions.json
- dialogues.json
- exercises.json
- review.json
- manifest.json

### Textbook bundle
- one markdown lesson chapter per unit
- review appendix
- glossary seed
- table of contents

### Flashcard / drill bundle
- vocab deck csv
- construction deck csv
- dialogue prompt deck csv

### Teacher bundle
- objectives
- pacing notes
- answer key
- teaching notes

## Exit criteria
A developer/editor can build:
- app prototype
- printable PDF/textbook draft
- flashcard imports
from bundle exports alone.
