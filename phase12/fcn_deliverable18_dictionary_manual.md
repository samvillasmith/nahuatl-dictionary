# Flor y Canto Nahuatl Dictionary
## Deliverable 18

**Project:** Flor y Canto Nahuatl  
**Version:** 0.1  
**Status:** Internal/public draft scaffold  
**Scope:** Dictionary design and publication template

---

## 1. Purpose

This deliverable defines the FCN dictionary as a publication-ready lexical system that connects:
- spoken EHN
- MSN neutral
- MSN poetic
- classical citation
- source tracking
- editorial status
- validation status

---

## 2. Entry Template

Each dictionary entry should be able to include:

- entry_id
- ehn_spoken_form
- msn_headword
- msn_poetic_form
- classical_citation_form
- part_of_speech
- register
- variety
- gloss_en
- gloss_es
- root_family
- source_reference
- source_confidence
- validation_status
- editorial_status
- notes_public
- notes_internal
- cross_references

---

## 3. Entry Directions

Users should be able to move:
- from EHN spoken form to neutral written form
- from neutral form to poetic form where allowed
- from modern form to classical citation note
- across lexical family / root family
- across variant or alternate forms

---

## 4. Cross-Reference Rules

Cross-reference types:
- synonym
- near-synonym
- root-family related
- register-related
- see-also poetic
- see classical citation
- see spoken form
- see neutral written form

Cross-references must not:
- silently collapse unlike registers
- hide that one form is classical-only
- erase variant or source differences

---

## 5. Test Dictionary Sample

A sample dictionary should include entries covering:
- body parts
- family
- household
- food
- daily verbs
- social interaction
- poetic items
- classical-note items

The sample should demonstrate:
- multi-register headword organization
- bilingual glossing
- source notes
- cross-references
- public-note clarity

---

## 6. Publication Form

The dictionary should be publishable in:
- database form
- web form
- app lookup form
- print/PDF form

The public version should favor:
- clean entry display
- intuitive cross-reference behavior
- clear register marking
- no hidden internal notes
