# register_conversion/ — Rules for Moving Between Registers

This directory documents how to convert text between the four FCN registers: EHN spoken, MSN neutral, MSN-P (poetic), and Classical citation. It provides explicit rules, worked examples, and a list of violations to avoid.

## What this directory contains

- Defines all four conversion directions with explicit rules
- Provides worked example pairs for each direction
- Documents the "what not to do" rules with risk labels
- Establishes the note-only policy for classical-only material

## The four conversion directions

| From | To | Operation | Key rule |
|---|---|---|---|
| EHN spoken | MSN neutral | Orthographic normalization + register shift | FCN conventions; no classical borrowing |
| MSN neutral | MSN-P | Elevation (device, diction, structure) | One device at a time; use approved inventory |
| Classical citation | MSN-P | Annotated adaptation | Every adaptation must retain its source annotation |
| Classical citation | Note-only | Preservation without reuse | Philological value; do not force into production registers |
