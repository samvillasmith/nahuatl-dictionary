# docs/ — Project Governance and Charters

This directory contains the foundational governance documents for Flor y Canto Nahuatl. These are the constitutional documents — they define what the project is, what it is committed to, and how decisions are made.

## Files

| File | Description |
|---|---|
| `Flor y Canto Nahuatl Source Hierarchy Document.md` | Source authority hierarchy — which sources govern which decisions, and how conflicts are resolved across registers |
| `founding_charter_v0.1.md` | Project founding charter — mission, scope, and founding principles |
| `register_charter_v0.1.md` | Register charter — formal definition of EHN, MSN neutral, and MSN-P, and the rules governing each |
| `mission_statements.md` | Project mission statements |
| `internal/core_premises.md` | Internal core premises |
| `internal/success_criteria.md` | Success criteria for the project |

## Read order for new contributors

1. `mission_statements.md` — understand what FCN is trying to accomplish
2. `founding_charter_v0.1.md` — understand the project's commitments and scope
3. `register_charter_v0.1.md` — understand the three-register model (EHN / MSN neutral / MSN-P)
4. `Flor y Canto Nahuatl Source Hierarchy Document.md` — understand how source conflicts are resolved
5. Then read `../reference_manuals/fcn_deliverable14_master_sourcebook.md` — the master sourcebook that synthesizes all of the above
